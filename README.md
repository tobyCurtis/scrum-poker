# scrum poker 

## Get started

Install the dependencies...

```bash
npm install
brew install deno
```

...then start the web dev server

```bash
npm run dev
```
...then start the deno websocket server

```bash
npm run deno
```

Before pushing to wherevr you're hosting, build with

```bash
npm run build
```
